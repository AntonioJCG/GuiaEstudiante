Pasen

¿Qué es PASEN?

PASEN es el módulo de Séneca que permite la comunicación entre los centros educativos y las familias, tutores legales y alumnado, ofreciendo una serie de funcionalidades, como: consulta de calificaciones, consulta y justificación de faltas de asistencia, datos del centro educativo, horario y calendario escolar, mensajería interna, etc.

Cómo obtener el usuario y la contraseña (autologin) de Pasen

El procedimiento de "autologin" está disponible para usuarios con perfiles tutor/a legal y alumnado (tanto emancipado como no). Mediante él, cualquier padre/madre o alumno/a podrá realizar un proceso de "autologin" para obtener las credenciales de acceso a PASEN (usuario y contraseña). Para realizar el "autologin" a través de la aplicación web o de la app iPasen, las familias deberán pinchar en el enlace 'No tengo acceso' de la pantalla de login de la página principal de Pasen:

http://portalseneca.ced.junta-andalucia.es/educacion/portalseneca/web/pasen/inicio.

Se requerirán tres datos: el NIF , una fecha de nacimiento (de algunos de sus hijos/as o la suya propia) y un número de teléfono móvil donde recibirá por SMS las credenciales. Este último deberá coincidir con el que consta en Séneca para los tutores legales o para el alumnado (consulte con el Instituto en caso de duda o necesidad de actualizar el número de teléfono).
